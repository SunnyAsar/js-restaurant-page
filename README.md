# Restaurant page
This is a restaurant page in JS featuring the use of webpack to compile modules.

### clone project

`
git clone https://github.com/SunnyAsar/js-restaurant-page
`

### go to project directory

`
### cd js-restaurant-page
`

### install dependencies

`
npm install
`

### start app 

`npm run dev `
